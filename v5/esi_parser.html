Option Name:esi_parser
Replaces:
Requires:--enable-esi
Default Value:Selects libxml2 if available at ./configure time or libexpat otherwise.
Suggested Config: 


	Selects the XML parsing library to use when interpreting responses with
	Edge Side Includes.

	To disable ESI handling completely, ./configure Squid with --disable-esi.
