## Option Name:

 * **Replaces:** 
 * **Requires:** 
 * **Default Value:** dns_timeout 30 seconds


## Suggested Config:


## Details:

	DNS Query timeout. If no response is received to a DNS query
	within this time all DNS servers for the queried domain
	are assumed to be unavailable.
