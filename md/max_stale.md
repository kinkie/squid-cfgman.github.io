 * **Replaces:** 
 * **Requires:** 
 * **Default Value:** max_stale 1 week


## Suggested Config:
```plaintext

```

## Details:

	This option puts an upper limit on how stale content Squid
	will serve from the cache if cache validation fails.
	Can be overridden by the refresh_pattern max-stale option.
