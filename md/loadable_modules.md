## Option Name:
[loadable_modules](#loadable_modules)
 * **Replaces:** 
 * **Requires:** --enable-shared
 * **Default Value:** none


## Suggested Config:
```plaintext

```

## Details:

	Instructs Squid to load the specified dynamic module(s) or activate
	preloaded module(s).
Example:
loadable_modules /usr/local/squid/lib/MinimalAdapter.so
